# Computer Vision Detection Projects

This repository contains a collection of **computer vision detection experiments** focused on facial analysis.  
The goal of these projects is to explore how machines can detect and understand **human facial features and expressions** using real-time camera input.

These projects were created for **learning, experimentation, and improving skills in AI and computer vision**.

---

## Features

The detection systems implemented include:

- **Face Detection** – Detects human faces in real-time using a webcam.
- **Eye Detection** – Identifies and tracks eyes within the detected face.
- **Facial Expression Detection** – Attempts to recognize expressions such as smiling.
- **Real-time Processing** – Runs directly from the camera feed.

---

## Technologies Used

- **Python**
- **OpenCV (cv2)** – Image processing and camera handling
- **MediaPipe** – Face and hand landmark detection
- **Machine Vision Techniques** for feature tracking and detection
